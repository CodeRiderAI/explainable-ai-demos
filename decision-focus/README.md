# Decision Focus — 視野と焦点化デモ

AIが「すべてを見ようとしない」設計を、動く映像で体験できるデモです。
視野半径・ターゲット数・速度を変えながら、「焦点化」の概念を体感できます。

---

## 🎮 操作方法

| 操作 | 内容 |
|------|------|
| ▶️ **Start / Pause / Reset** | シミュレーションの開始・停止・リセット。 |
| 🖱️ **視点をマウス追従** | チェックを入れるとマウス位置にフォーカスが移動。 |
| 🎚️ **視野半径 / ターゲット数 / 速度** | 観測範囲・対象数・動作速度をリアルタイムに調整。 |
| 💾 **ログを保存** | 現在の焦点化データをJSON形式でダウンロード。 |

---

## 🧠 コンセプト

- **フレーム問題（Frame Problem）** の可視化。  
  → すべてを考慮するのではなく、「見える範囲で最適化」する。  
- **焦点化（Focus）**  
  → 周囲の情報密度に応じて、視野内の注目領域を自律的に更新。  
- **視野と注目の分離**  
  → “見えるもの” と “考慮するもの” の境界を明示。
